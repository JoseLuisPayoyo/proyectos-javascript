# 🚀 Proyectos del Curso JavaScript Moderno (Udemy)

Este repositorio contiene todos los proyectos desarrollados como parte del curso [JavaScript Moderno: La Guía Definitiva, Construye +20 Proyectos] impartido en Udemy.

A lo largo del curso se trabajan fundamentos y técnicas modernas de JavaScript aplicadas al desarrollo de proyectos reales, sin frameworks, para fortalecer el dominio del lenguaje y el pensamiento lógico.

## 🎯 ¿Qué encontrarás aquí?

- Aplicaciones 100% JavaScript puro (sin frameworks)
- Buenas prácticas modernas del lenguaje
- Proyectos incrementales desde nivel básico hasta avanzado
- Enfoque en manipulación del DOM, eventos, asincronía y consumo de APIs

## 🧩 Proyectos incluidos

1. Cotizador de Criptomonedas
2. Buscador de Recetas
3. Administrador de Citas
4. Buscador de Imagenes
5. CRUD de Clientes IndexedDB
6. Localizador de Clima
7. Generador de Frases
8. Validador de Formularios
9. Interfaz de Noticias
10. Y más...
